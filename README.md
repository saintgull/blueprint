# The Blueprint Magazine

A clean, focused website for The Blueprint Magazine - transforming grassroots projects into pathfinders for Network Governance.

## Overview

The Blueprint Magazine collects and shares methodologies from community-led projects, creating a knowledge base that helps increase the success rate of grassroots initiatives by 30%+.

## Features

- **Responsive Design**: Clean, modern layout that works on all devices
- **Visual Hierarchy**: Prominent mission statement and clear navigation
- **3-Phase Roadmap**: Visual progression from Magazine → Data & Tools → Ecosystem
- **Submission Guidelines**: Dedicated page for writers with 6 content categories
- **Interactive Elements**: Hover effects and smooth transitions

## Technical Details

- **Colors**: Primary blue (#1237b2) with 40% opacity dotted grid background
- **Typography**: 
  - Headers: Libre Baskerville (serif)
  - Body: Montserrat (sans-serif)
- **Navigation**: Fixed banner with 60% opacity and backdrop blur
- **Layout**: Maximum width 900px, centered with responsive grid systems

## Pages

1. **Home** (`index.html`) - Main landing page with mission, problem, roadmap, and CTAs
2. **For Writers** (`for-writers.html`) - Submission guidelines and content categories
